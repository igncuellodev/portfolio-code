Nacho | Design Engineer Portfolio
This is my personal space documenting the intersection of systematic design and software engineering. As a Design Engineer, my focus is on building interfaces where logic and aesthetics converge to create scalable and efficient products.

🛠 Tech Stack
I have selected tools that prioritize performance and a fluid user experience:

Vite: Next-generation development engine for near-instant loading.

Tailwind CSS: Utility-first styling framework to maintain visual consistency.

Lenis Scroll: Smooth scroll implementation for a premium navigation feel.

Optimized SVGs: Hand-polished assets to minimize web weight and improve speed.

🚀 Project Philosophy
Visual Integrity: Maintaining total fidelity between creative vision and technical execution.

Clean Architecture: Organized, modular, and easy-to-maintain codebase.

Purposeful Motion: Animations and transitions designed to enhance usability, not distract from it.

💻 Local Setup
To explore the project on your own machine:

Clone this repository.

Install dependencies using npm install.

Start the development server with npm run dev.

View the result at the local port indicated in your terminal.
